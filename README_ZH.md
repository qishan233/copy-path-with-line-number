# copy-path-with-line-number

[英文介绍](https://github.com/qishan233/copy-path-with-line-number/blob/main/README.md)

此扩展提供了在资源管理器、标签标题和编辑器上下文中复制文件路径和行号的功能。

它支持自定义文件路径分隔符、范围连接符和选择分隔符。

它还支持中文和英文（安装此扩展后需要重新加载窗口以激活效果）。
